如果你是一个Emby用户，你可能会遇到这样的问题：  
虽然你找到了一整季匹配的字幕文件，但是它们的文件名与视频文件的文件名不匹配。  
这种情况下，Emby就无法自动将字幕文件与正确的视频文件对应起来，这让一整季的字幕文件都需要手动重命名，非常繁琐。  
依靠着我没有几页的Python知识写了这个脚本，可以将字幕文件名字进行批量重命名，来减少一些繁琐的工作
